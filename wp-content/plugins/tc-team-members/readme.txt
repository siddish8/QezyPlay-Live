== Installation ==

Installing  TC Team Members is easy like  a regular WordPress plugin. Here goes the step by step Guide to Install TC Team Members plugin :

= Install Plugin using WordPress Admin Panel Search Option =
- In your Admin, go to menu Plugins > Add New
- Search TC Team Members
- Click to install
- Activate the plugin
- TC Team Members post type will show below POSTS .

= Install Plugin using Upload Method in WordPress Admin Panel =

- Download the latest version of TC Team Members (.zip file)
- In your Admin, go to menu Plugins > Add New
- click on the Upload Tab
- Click on “Choose File from your Hard Disk”
- Browse and Select downloaded tc-team-members.zip & click on ‘Install Now’ button
- Activate the plugin
- TC Team Members type will show below POSTS

= Install  Plugin using FTP Manager =
- Upload TC Team Members folder with all files inside the /wp-content/plugins/ directory
- Go to WordPress dashboard > Plugins & Activate the  TC Team Members plugin
- TC Team Members post type will show below POSTS with its owl icon.
