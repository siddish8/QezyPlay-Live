myStickyElements is simple yet very effective plugin. It is perfect to fill out usually unused side space on webpages with some additional messages
== Description ==
Using the myStickyElements administration backend you can add up to 10 elements, position them left or right, change vertical position, change icon img, background color, content and more...

Plugin Home: http://wordpress.transformnews.com/plugins/mystickyelements-simple-sticky-side-elements-for-wordpress-504
